# 🎶 Web Bandas Musicais 🎸

Bem-vindo à **web das bandas musicais**! 🚀 Aqui, podes explorar as tuas bandas favoritas, descobrir álbuns incríveis e manter tudo organizado num só lugar. 😎 Adiciona, edita ou elimina bandas e álbuns enquanto desfrutas de uma interface super intuitiva e fácil de usar. 🤩

## 🚀 Funcionalidades  

✅ **Adicionar Bandas** – Regista novas bandas com foto, descrição e um botão direto para o YouTube. 🎤  
✅ **Adicionar Álbuns** – Mantém organizada a discografia das tuas bandas favoritas. 💿  
✅ **Contador de Álbuns** – Visualiza quantos álbuns cada banda tem. 📊  
✅ **Edição de Informação** – Atualiza os detalhes das bandas e álbuns com facilidade. ✏️  
✅ **Eliminar Bandas ou Álbuns** – Remove qualquer conteúdo com um clique. ❌  
✅ **Link para o YouTube** – Acede ao canal oficial da banda em apenas um toque. 🎥🎶  
✅ **Autenticação de Utilizadores** – Registo, login e logout seguros. 🔐  
✅ **Recuperação de Senha** – Se esqueceste a senha, recupera-a através do email usando Mailtrap! 📧  
✅ **Proteção com Middleware** – Áreas restritas da web protegidas contra acessos não autorizados. 🔒
